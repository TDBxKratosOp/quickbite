# 🍔 QuickBite — Real Food, Fast

A modern full-stack food delivery web app built with React, Firebase, and Razorpay.

---

## ✨ Features

- 🔐 **Authentication** — Signup & Login with Firebase Auth
- 🍽️ **Restaurant Browsing** — Browse restaurants and menus
- 🛒 **Cart System** — Add, remove, and manage cart items
- 💳 **Razorpay Payments** — Secure online payments + Cash on Delivery
- 🏷️ **Coupon System** — Apply discount coupons stored in Firestore
- 📦 **Order Tracking** — Real-time order status updates
- ⭐ **Reviews** — Leave reviews for delivered orders
- 👤 **Profile Management** — Update name, address, pincode
- 🌙 **Dark Mode** — Full dark/light theme support
- 📱 **Fully Responsive** — Works on mobile and desktop

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React 19, TypeScript, Tailwind CSS |
| Backend | Firebase (Auth + Firestore) |
| Payments | Razorpay |
| Hosting | Netlify + Netlify Functions |
| Build Tool | Vite |
| Animations | Framer Motion |
